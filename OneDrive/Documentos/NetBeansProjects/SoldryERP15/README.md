\# SoldryERP



Sistema ERP para gestión de inventarios, clientes, facturación y proveedores.



\## Tecnologías

\- Java

\- MySQL

\- NetBeans

\- JDBC



\## Funcionalidades

\- Conexión a base de datos

\- Insertar productos

\- Listar productos

\- Actualizar productos

\- Eliminar productos

\- CRUD completo



\## Base de datos



CREATE DATABASE soldryerp;



CREATE TABLE inventario(

&nbsp; id INT AUTO\_INCREMENT PRIMARY KEY,

&nbsp; nombre VARCHAR(100),

&nbsp; cantidad INT,

&nbsp; precio DOUBLE

);
